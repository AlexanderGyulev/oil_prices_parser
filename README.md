# Oil Price Lookup Application

## Description

This is a simple desktop application built with Python and Tkinter that allows users to look up historical crude oil prices by date.

The application loads oil price data from an XML file and provides a minimal graphical interface where the user can enter a date and retrieve the corresponding price.

---

## Features

- Load oil price data from an XML file
- Simple and intuitive graphical user interface
- Search oil prices by date (`YYYY-MM-DD`)
- Handling of missing data

---

## Technologies Used

- Python 3
- Tkinter (GUI framework)
- xml.etree.ElementTree (XML parsing)

---

## Running the Application Locally

### Clone the repository

```bash
git clone https://github.com/your-username/oil-price-app.git
cd oil-price-app
```

### Run the application

```bash
python main.py
```
